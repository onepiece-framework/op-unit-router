Router
===

 Router is onepiece-framework's unit.

# Usage

# Methods

## EndPoint

 Get end-point file path.

```php
$endpoint = \OP\Unit::Singleton('Router')->EndPoint();
```

## Args

 Get SmartURL arguments.

```php
$args = \OP\Unit::Singleton('Router')->Args();
```
